Detailed Description

template<uint64_t MAX_BIT_LENGTH = 8092>
class stool::NaiveBitVector< MAX_BIT_LENGTH >

A simple bit vector B[0..n-1] implementation with push/pop operations.

Note
The bits B[0..n-1] are stored in 64-bit integers S[0..m-1] (uint64_t *buffer_)
Template Parameters
MAX_BIT_LENGTHMaximum number of bits that can be stored (default: 8092)
