Detailed Description

A structure to store LCP (Longest Common Prefix) information [Unchecked AI's Comment].

LCPInfo stores both the LCP value and its corresponding position in the text. It includes:

  • LCP value storage
  • Position tracking
  • Default and parameterized constructors

The structure is particularly useful for:

  • LCP array construction algorithms
  • Suffix array based text processing
  • Pattern matching applications
Note
This implementation uses uint64_t for both LCP values and positions to handle large texts efficiently
