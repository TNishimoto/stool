Checks if a text T[0..n-1] contains a end marker character c.

Template Parameters
CHARThe character type of the text
Returns
true if the text satisfies all conditions, throws an exception otherwise
Exceptions
std::logic_errorif any of the conditions are violated
Note
The character c must satisfy the following conditions: (i) any character in T[0..n-2] is smaller than c and (ii) the last character of T is c.
