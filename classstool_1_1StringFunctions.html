inlinestatic

Checks if a text contains a special marker character.

This function verifies that a given text contains a special marker character 'c' only at the end of the text. It checks three conditions:

  1. The marker character appears exactly once at the end of the text
  2. No characters in the text are less than the marker character
  3. The marker character does not appear anywhere else in the text
Template Parameters
CHARThe character type of the text
Parameters
textThe input text to check
cThe special marker character to look for
Returns
true if the text satisfies all conditions, throws an exception otherwise
Exceptions
std::logic_errorif any of the conditions are violated
