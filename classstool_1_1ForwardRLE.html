Detailed Description

template<typename TEXT_ITERATOR_BEGIN, typename TEXT_ITERATOR_END, typename CHAR = uint8_t>
class stool::ForwardRLE< TEXT_ITERATOR_BEGIN, TEXT_ITERATOR_END, CHAR >

Forward iterator for Run-Length Encoding (RLE) of text [Unchecked AI's Comment].

This class provides a forward iterator that traverses text and groups consecutive identical characters into runs. It implements run-length encoding on-the-fly without storing the entire encoded sequence in memory.
