Generates all possible permutations of a given length using the provided alphabet.

Parameters
lenThe length of each permutation to generate.
alphabetThe set of characters to use for permutations.
Returns
A vector containing all possible permutations of the specified length.

This function generates all possible combinations of characters from the alphabet that have the specified length. For example, with alphabet {'a', 'b'} and length 2, it would generate {"aa", "ab", "ba", "bb"}.

static std::vector< uint8_t > stool::StringGenerator::create_uint8_t_string ( std::vector< uint32_t > &  original_text)
inlinestatic

Converts a sequence of integers to a string using an 8-character alphabet.

Parameters
original_textThe input sequence of integers.
Returns
A vector of uint8_t characters representing the converted string.

The function maps each integer to one of 8 characters: 'a', 'b', 'c', 'd', 'e', 'f', 'g', 'h'. The mapping is done using modulo 8 operation on each integer.
