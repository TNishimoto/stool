Detailed Description

A class for computing partial sums (psum) query on packed integer sequences.

This class provides functions for computing psum query on a k-bit integer sequence S = s_{0}, s_{1}, ..., s_{n-1} packed into a 64 bit integer vector B[0..nk-1]. Here, s_{i} is represented as B[i*k..(i+1)*k-1] for all i. Psum query returns the sum of the first m elements of S for a given m.
