Detailed Description

A small helper component for Beller's LCP array construction algorithm [Unchecked AI's Comment].

BellerSmallComponent provides auxiliary functionality for the main Beller algorithm implementation. It includes:

  • Tracking of last processed indices and bounds
  • Occurrence flag management
  • Character occurrence set maintenance

The class is particularly useful for:

  • State tracking during LCP computation
  • Character occurrence bookkeeping
  • Efficient interval bound management
Note
This implementation is designed to work closely with BellerComponent for efficient LCP array construction
