Finds the position of the (i+1)-th set bit (1) in a 64-bit integer.

Return the position of the (i+1)-th 1 in the given bits if such bit exists; otherwise return -1.

Parameters
bitsThe 64-bit integer to search
iThe zero-based index of the set bit to find (finds i+1 th occurrence)
Returns
The position (0-63) of the (i+1)-th set bit, or -1 if not found
