Detailed Description

A class for supporting search query on packed integer sequences.

This class provides search query on a k-bit integer sequence S = s_{0}, s_{1}, ..., s_{n-1} packed into a 64 bit integer vector B[0..nk-1]. Here, s_{i} is represented as B[i*k..(i+1)*k-1] for all i. Search query returns the smallest position i such that s_{0} + s_{1} + ... + s_{i} >= x for a given x.
