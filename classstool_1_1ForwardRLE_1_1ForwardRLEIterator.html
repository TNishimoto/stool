Detailed Description

template<typename TEXT_ITERATOR_BEGIN, typename TEXT_ITERATOR_END, typename CHAR = uint8_t>
class stool::ForwardRLE< TEXT_ITERATOR_BEGIN, TEXT_ITERATOR_END, CHAR >::ForwardRLEIterator

Iterator class for traversing RLE runs.

This iterator provides access to consecutive runs of identical characters in the text. It implements the standard forward iterator interface.
