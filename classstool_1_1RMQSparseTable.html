Detailed Description

template<typename T = uint64_t>
class stool::RMQSparseTable< T >

Range Minimum Query (RMQ) implementation using Sparse Table data structure [Unchecked AI's Comment].

This class provides efficient range minimum query operations on an array using the sparse table technique. It preprocesses the array in O(n log n) time and can answer RMQ queries in O(1) time.

Template Parameters
TThe data type of the array elements (default: uint64_t)
