Detailed Description

template<bool USE_PSUM = true>
class stool::NaiveFLCVector< USE_PSUM >

A naive vector implementation using fixed-length codes for non-negative integer sequences S[0..n-1].

Note
The non-negative integer sequence S[0..n-1] is stored in the 64-bit integers buffer B[0..m-1]. Each integer of S is encoded as a fixed-length code of bit length 2^{x} for an integer x in { 1, 2, 4, 8, 16, 32, 64 }.
Template Parameters
USE_PSUMBoolean parameter to enable/disable prefix sum maintenance
