Detailed Description

template<uint64_t MAX_BIT_LENGTH = 8092>
class stool::BitArrayDeque< MAX_BIT_LENGTH >

A simple circular buffer-based deque implementation.

Template Parameters
TThe type of elements stored in the deque
INDEX_TYPEThe type used for indexing (default: uint16_t)

This class provides a memory-efficient deque implementation using a circular buffer. It supports O(1) push/pop operations at both ends and random access to elements. The buffer size is automatically managed to maintain optimal memory usage.
